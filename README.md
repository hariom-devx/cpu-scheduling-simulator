# CPU Scheduling Simulator

A web-based CPU Scheduling Simulator built using Python and Streamlit.

## 🔧 Features
- FCFS
- SJF (Non-Preemptive)
- SRTF (Preemptive)
- Round Robin
- Priority Scheduling

## 📊 Outputs
- Gantt Chart
- Waiting Time (WT)
- Turnaround Time (TAT)
- Response Time (RT)
- Algorithm Comparison

## ▶️ Run Locally

pip install -r requirements.txt  
streamlit run app.py

## 🌐 Deployment
Deployed using Streamlit Cloud.
